Custom Animation

Create your own custom animations using the ModalAnimations.create() method:

const customAnimation = ModalAnimations.create({ show: async (modal, triggerElement) => { // Your custom show animation logic const container = modal.elements.container; container.style.opacity = '0'; requestAnimationFrame(() => { container.style.opacity = '1'; }); return new Promise(resolve => setTimeout(resolve, 300)); }, hide: async (modal) => { // Your custom hide animation logic const container = modal.elements.container; container.style.opacity = '0'; return new Promise(resolve => setTimeout(resolve, 300)); } }); const modal = new Modal({ title: 'Custom Animation', body: '

Using custom animation!

', animation: customAnimation });

Chaining Animations

Chain multiple animations to create complex sequences:

const chainedAnimation = ModalAnimations.chain( ModalAnimations.zoom, ModalAnimations.fade ); const modal = new Modal({ title: 'Chained Animation', animation: chainedAnimation });

Parallel Animations

Run multiple animations simultaneously:

const parallelAnimation = ModalAnimations.parallel( ModalAnimations.fade, ModalAnimations.zoom ); const modal = new Modal({ title: 'Parallel Animation', animation: parallelAnimation });
