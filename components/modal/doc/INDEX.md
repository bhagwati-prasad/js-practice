## üé¨ Animation Quick Reference

### All Available Animations
1. **fromElementToCenter** - Opens from trigger element (default)
2. **fade** - Simple fade in/out
3. **slideFromTop** - Slides down from top
4. **slideFromBottom** - Slides up from bottom
5. **zoom** - Scales up with bounce
6. **flip** - 3D flip effect
7. **rotateScale** - Rotate and scale
8. **none** - No animation

### Animation Methods
- `ModalAnimations.create()` - Create custom animation
- `ModalAnimations.chain()` - Chain animations sequentially
- `ModalAnimations.parallel()` - Run animations together
